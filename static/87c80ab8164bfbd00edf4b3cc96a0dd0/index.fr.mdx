Bienvenue sur ce site vous permettant de développer des applications web modernes à l'aide de React, Node.js & MongoDB.

N'hésitez pas à commencer en découvrant l'introduction [**ici**](/intro).

Les contenus du cours contiennent certains éléments graphiques qui reviennent régulièrement. Voici leurs définitions :
- 🍬 : cela signifie que vous pouvez consulter ce contenu de manière optionnelle, c'est un bonus. Les contenus tagués par un 🍬 sont associés soit :
    - à de l'approfondissement de concepts ;
    - à l'acquisition de prérequis à ce cours-ci ;
    - à l'ouverture vers des compétences qui ne sont pas développées dans ce cours-ci.
- 💭 : cela signifie que vous devriez offrir un peu de votre temps de réflexion avant de continuer à lire la suite ; lorsque vous réalisez un tutoriel, les temps de réflexion tagués par un 💭 sont essentiels pour challenger vos acquis ; évitez donc de lire tous les contenus d'une seule traite.
- 👍 : cela signifie qu'une bonne pratique est mise en avant dans le texte qui suit le 👍.
- 👎 : cela veut dire qu'une mauvaise pratique est soulignée dans le texte qui suit le 👎.
- ⚡ : le texte qui suit un ⚡ décrit un danger habituellement rencontré par les programmeurs.
